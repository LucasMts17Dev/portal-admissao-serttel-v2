# Portal de Admissão Serttel

Este projeto é uma base para o portal de admissão da Serttel, com:
- tela inicial com os cards de Nova Admissão, Documentos e Compliance;
- navegação entre Documentos e Auditoria;
- fluxo de upload e análise de PDFs com preview textual simples.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
